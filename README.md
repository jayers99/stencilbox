# Development Templates

Reusable templates and guard rails for common development patterns.

## Contents

### python-cli-scaffold/
Template for scaffolding new Python CLI projects with:
- Modern Software Engineering principles (Dave Farley)
- Test-Driven Development (TDD) with pytest
- Domain-Driven Design (DDD) architecture
- Cloud-focused tooling (AWS/GCP)
- Trunk-based development workflow

See [python-cli-scaffold/guard-rails-prompt.md](python-cli-scaffold/guard-rails-prompt.md) for the full template.

## Usage

Copy the appropriate guard rails prompt into Claude Code and fill in your project-specific details.
