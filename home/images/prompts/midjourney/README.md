## About Midjourney

Midjourney excels at:
- Artistic and aesthetic styles
- Composition and visual appeal
- Parameter-based fine control
- Consistent style generation

## Getting Started

1. Use keywords and specific parameters (`--ar`, `--v`, `--style`)
2. Always specify aspect ratio and version
3. Use variations (V1-V4) and remix mode to iterate
4. For transparency needs, see the quick reference guide above
5. Document successful prompts with full parameters

## Essential Parameters

```
--ar 16:9    # Aspect ratio
--v 6        # Version
--style raw  # Less Midjourney aesthetic
--no [item]  # Negative prompt
--q 2        # Quality
--s 250      # Stylize (0-1000)
```
