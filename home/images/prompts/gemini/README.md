Prompts and guides for Google's Gemini/Imagen and Nano Banana Pro image generation.

- [Transparent Background Guide](transparent-background-quick-ref.md) - Essential for web graphics, stickers, and logos
- [Example: Logo Design](example-logo-transparent.md) - Complete workflow with verification steps

## About Gemini / Nano Banana Pro

Gemini excels at:
- Fast generation speed
- Integration with Google ecosystem
- Conversational refinement
- Clear, specific descriptions

## Getting Started

1. Use clear, specific descriptions
2. Iterate and refine through conversation
3. Explain your use case for better results
4. For transparency needs, see the quick reference guide above
5. **Always verify alpha channel** - Gemini may produce fake transparency
6. Document successful prompts for reuse

## Transparency Warning

⚠️ Gemini sometimes produces **checkerboard patterns** that look like transparency but are baked into the image. Always:
- Check file properties for alpha channel
- Test on different colored backgrounds
- Use image editor to verify true transparency
- See quick reference guide for solutions
