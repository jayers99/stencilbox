# DALL-E Prompts

Prompts and guides for ChatGPT's DALL-E image generation.

## Quick References

- [Transparent Background Guide](transparent-background-quick-ref.md) - Essential for web graphics, stickers, and logos
- [Example: Web Banner](example-web-banner-transparent.md) - Complete workflow for transparent web banner

## About DALL-E

DALL-E excels at:
- Natural language understanding
- Photorealistic images
- Text within images
- Conversational iteration

## Getting Started

1. Use conversational, detailed descriptions
2. Iterate with "Make it more..." or "Keep X but change Y"
3. For transparency needs, see the quick reference guide above
4. Document successful prompts for reuse

## Contributing

When you create a prompt that works well:
1. Save it using the format from [Image_Human_AI_Team_Agreement.md](../../Image_Human_AI_Team_Agreement.md#8-prompt-documentation-format)
2. Include the prompt text, what worked, and when to use it
3. Tag with relevant keywords
